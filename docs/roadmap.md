# 学习路线与考核说明

## 学习路线

本课程共分为 **六大模块**，逐步深入，从基础知识到企业级应用开发，再到智能化功能集成，最终通过综合项目将所学知识付诸实践。以下是本课程的详细学习路线：

### 1. Web 开发基础与效能工具
- **目标**：帮助学生快速建立 Web 开发环境，熟悉必要的开发工具。
- **重点内容**：
  - 理解 **B/S 架构** 与 **C/S 架构** 的区别。
  - 学习 **Maven 构建工具**，掌握项目依赖管理。
  - 使用 **Git** 进行版本控制和代码管理。
  - 配置 **IDEA** 开发环境，快速启动项目。
  - 学习 **AI 辅助编程工具**（如通义灵码/Cursor）提升开发效率。
- **学习方法**：理论与工具操作结合，实践中快速配置开发环境并进行初步的代码编写。

### 2. Web 底层原理
- **目标**：深入理解 Web 开发的基础原理，为后续开发奠定理论基础。
- **重点内容**：
  - 学习 **HTTP 协议** 的核心知识，掌握请求、响应、状态码等概念。
  - 理解 **Session 与 Cookie** 的使用，掌握用户登录与会话管理。
  - 学习 **Servlet 生命周期** 和 **Java Web 工作原理**（仅了解基本概念）。
  - JSON 数据格式的应用与解析，使用 **Jackson** 进行 JSON 处理。
- **学习方法**：结合理论与实际，进行 HTTP 请求与响应模拟，掌握基本的 Web 开发原理。

### 3. Spring Boot 快速开发
- **目标**：快速掌握 Spring Boot 框架，能够独立开发简单的 Web 应用。
- **重点内容**：
  - 理解 **Spring Boot** 的设计思想，学会快速构建和配置 Spring Boot 应用。
  - 深入了解 **IoC / DI** 原理，掌握依赖注入与控制反转。
  - 熟悉 **RESTful API** 设计，掌握接口规范与统一响应结构。
  - 使用 **Postman** 或 **Apifox** 进行接口测试。
- **学习方法**：通过搭建简单的 RESTful API 项目，理解并实践 Spring Boot 的核心特性。

### 4. 数据持久化与信创环境
- **目标**：掌握数据库操作技术，理解如何与数据库进行交互，使用国产数据库进行开发。
- **重点内容**：
  - 学习 **MyBatis** 框架，理解 ORM 原理与数据库操作。
  - 通过 **openGauss**（国产数据库）进行数据库操作，连接与配置数据库。
  - 学习如何进行 **事务管理**，保障数据一致性。
  - 掌握数据库的 **增删改查**（CRUD）操作。
- **学习方法**：进行数据库操作练习，通过 MyBatis 与 openGauss 完成数据管理任务。

### 5. AI 集成与外部接口调用
- **目标**：了解如何将 AI 大模型接口集成到 Web 应用中，实现智能化功能。
- **重点内容**：
  - 使用 **RestTemplate** 或 **WebClient** 进行 HTTP 客户端编程，调用外部 API。
  - 掌握 **AI 大模型 API** 的调用，如 **DeepSeek** 或 **OpenAI**。
  - 学习如何构建 **Prompt 工程**，优化与 AI 的交互。
  - 实现 **AI 旅行助手** 功能，通过 AI 推荐旅行目的地。
- **学习方法**：结合实际需求，集成 AI 接口，增加 Web 应用的智能功能。

### 6. 综合项目：智能航班管理系统实战
- **目标**：通过综合项目，将所有学到的知识融合，完成一个完整的 Web 系统。
- **重点内容**：
  - 系统设计与架构：E-R 图、数据库设计与接口设计。
  - 模块开发：用户登录、航班查询、预订等业务逻辑实现。
  - AI 模块集成：根据目的地自动生成游玩攻略，提升用户体验。
  - 系统部署与上线：通过 Maven 打包与 Linux 服务器部署，完成系统的上线与演示。
- **学习方法**：项目驱动式学习，通过团队协作，完成项目的需求分析、开发、部署和展示。

## 考核说明

### 平时成绩（50%）
- **课堂互动与考勤（10%）**：鼓励学生在课堂上提问，积极参与讨论。
- **实验报告（20%）**：每个实验完成后需要提交代码和报告，报告包括实验过程、问题与解决方案等。
- **AI 应用报告（20%）**：学生需提交一份关于 **AI API 集成** 的心得报告，内容包括遇到的技术问题、解决方法及应用场景等。

### 期末项目考核（50%）
- **期末项目（50%）**：学生需完成 **智能航班管理系统** 的开发与演示，项目包括以下几个部分：
  - 基础功能：用户登录、航班查询与预订等。
  - 信创环境：成功连接并操作 **openGauss** 数据库。
  - AI 功能：成功调用 AI 大模型接口，生成旅行计划或相关功能。
  - 项目答辩：在项目展示后，进行答辩，解释项目设计、实现过程及技术选择。
- **评分标准**：
  - 基础功能（60%）：包括登录、查询、预订等基本功能。
  - 信创环境（10%）：使用 openGauss 数据库，符合国产化要求。
  - AI 功能（30%）：成功集成 AI 大模型 API，并实现智能功能。

## 学习建议

- **预备知识**：建议学生在课程开始前复习 **Java 编程**、**数据库原理**、**计算机网络** 等相关课程。
- **学习方法**：
  - 理论与实践结合，确保每个模块都进行相应的代码实践。
  - 课程中的实验和项目是关键，务必按照课程要求完成实验和项目。
  - 积极参与课堂互动，及时解决学习过程中遇到的疑难问题。
