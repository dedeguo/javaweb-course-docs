# 《Java Web 开发技术》课程大纲

本课程为计算机科学与技术专业的专业必修课程之一，旨在让学生掌握 Web 应用开发的基础知识与关键技术，为后续学习 Spring、微服务架构、全栈开发等内容打下坚实基础。

---

## 一、课程基本信息

| 项目 | 内容 |
|------|------|
| 课程名称 | Java Web 开发技术 |
| 课程类别 | 专业必修课 |
| 学时 | 48 学时（32 理论 + 16 实验） |
| 课程负责人 | Chende |
| 适用专业 | 计算机科学与技术 |
| 先修课程 | Java 程序设计、数据库原理 |

---

## 二、课程简介

本课程以 Java 技术体系为基础，介绍 Web 应用开发的基本概念、运行机制、服务端组件、应用架构与项目开发实践。通过本课程的学习，学生能够掌握 Web 开发的核心技术体系，包括 **Servlet、JSP、EL、JSTL、Filter、Listener、MVC 模型** 等，并能利用这些技术构建一个完整的 Java Web 应用。

## 四、教学内容与学时分配

| 章节 | 内容 | 学时 |
|------|------|------|
| 第1章 Web 基础与 HTTP | Web 架构、URL、HTTP 请求/响应、状态管理概念 | 4 |
| 第2章 Servlet 技术 | Servlet 生命周期、请求转发、重定向、参数处理 | 8 |
| 第3章 JSP 技术 | JSP 指令、内置对象、EL 表达式、JSTL 标签库 | 8 |
| 第4章 MVC 模式与三层架构 | MVC 理念、Controller/Service/DAO 分层设计 | 6 |
| 第5章 过滤器与监听器 | Filter、Listener 的运行机制与使用场景 | 4 |
| 第6章 会话管理 | Cookie、Session、Token、登录验证案例 | 4 |
| 实验课内容 | 环境搭建、Servlet/JSP 实验、MVC 实验、课程项目开发 | 16 |

---

## 五、实验/实践教学内容

实验重点围绕：

- Tomcat 环境搭建
- Maven Web 项目构建
- Servlet 实验：表单提交、请求转发/重定向
- JSP + EL/JSTL 实验
- MVC 案例实践
- 会话管理与登录案例
- 小型综合项目：如“在线留言板”“简易图书管理”等

学生将完成：

- 代码实现  
- 调试日志  
- 实验报告  
- 课程期末项目

## 六、教学资源与工具

- 开发环境：JDK 17/21、Tomcat 9/10/11
- 开发工具：IntelliJ IDEA / VS Code

## 七、学习方法建议

- 理解 HTTP、Servlet 生命周期等底层机制  
- 实践优先，多做实验、多调试  
- 课后复习示例代码  
- 使用 Git 管理代码版本  
- 遇到问题时善用 Postman、浏览器开发者工具、日志调试

## 九、课程项目（Course Project）

学生需要基于 MVC 架构完成一个小型 Web 项目，包括但不限于：

- 在线留言板系统  
- 图书借阅管理系统  
- 用户注册与登录模块  
- 简易博客系统  

项目内容需包含：

- 需求文档  
- 系统设计（流程图、E-R 图、表结构）  
- 源代码  
- 运行截图与部署说明

## 十、课程预期成果

完成课程后，学生应具备：

- 能独立开发基于 Java 的 Web 应用  
- 理解企业级 Java Web 架构思维  
- 熟悉常用开发工具与调试工具  
- 具有一定项目实践经验
