# 综合项目导读｜智能航班管理系统

> 以真实应用为驱动，整合前五章能力：用户认证、航班管理、AI 助手、部署演示与答辩治理。

## 项目目标

- 按模块完成用户/航班/AI 功能并联调。
- 在国产数据库 openGauss 上稳定运行。
- 打通从本地开发到部署演示的全链路。

## 里程碑

- M1：项目脚手架与基础路由完成。
- M2：数据库建模与 MyBatis CRUD 完成。
- M3：AI 助手能力接入与前端联调。
- M4：部署文档、演示脚本与答辩材料。

## 验收标准（建议）

- 功能正确：核心用例覆盖且无高危缺陷。
- 代码质量：模块清晰、接口与异常处理规范。
- 文档完备：README、部署说明、接口文档与演示脚本。
