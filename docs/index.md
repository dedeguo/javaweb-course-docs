# Java Web开发技术 · 课程首页

&gt; **"Spring Boot优先，AI赋能，信创立项"** —— 48学时实战驱动型课程

---

## 🎯 课程定位

本课程是计算机科学与技术专业的**专业核心必修课**，专为信创产业需求和AI时代开发效能升级而设计。课程颠覆传统"Servlet→JSP→SSM"的缓慢学习曲线，采用 **"Spring Boot First"** 激进策略，带领学生在48学时内完成从接口设计、国产数据库交互到AI智能化功能实现的全栈开发能力构建。

---

## ✨ 三大核心特色

### 1️⃣ **技术栈激进现代化**
- **零XML配置**：跳过繁琐的SSM整合，直接拥抱Spring Boot约定大于配置
- **信创生态实战**：全程基于龙蜥操作系统 + openGauss国产数据库
- **AI原生开发**：将通义灵码/Cursor等AI编程助手作为标配生产力工具

### 2️⃣ **项目驱动式学习**
以 **"智能航班管理系统"** 贯穿始终，每个实验都在同一项目中迭代：
- 用户模块：Session登录与RESTful认证
- 航班模块：openGauss数据库CRUD与余票扣减
- **AI智能助手**：集成DeepSeek API实现目的地攻略自动生成

### 3️⃣ **考核反内卷设计**
- **无闭卷考试**：期末考核=项目答辩+代码审查
- **AI应用报告**：考核Prompt工程与JSON解析实战能力，拒绝死记硬背
- **Git提交画像**：通过Commit频率和质量评估真实参与度

## 🛠️ 核心技术栈

| 层级 | 技术方案 | 说明 |
| :--- | :--- | :--- |
| **操作系统** | 龙蜥OS (Anolis) | 信创环境适配 |
| **编程语言** | Java 8+ | 面向对象基础要求 |
| **构建工具** | Maven | 依赖管理与项目打包 |
| **Web框架** | Spring Boot 2.x+ | 快速开发RESTful API |
| **数据库** | openGauss | 华为开源国产数据库 |
| **ORM框架** | MyBatis | SQL映射与事务控制 |
| **AI大模型** | DeepSeek API | 智能旅行助手功能 |
| **开发工具** | IntelliJ IDEA + 通义灵码/Cursor | AI辅助编程 |

## 📚 课程目标

通过本课程，你将能够：

✅ **理解核心原理**：B/S架构、HTTP协议、Session机制、Spring IoC/DI与AOP思想  
✅ **掌握工程能力**：独立使用Spring Boot+Mybatis开发企业级RESTful API，熟练使用AI工具提升3倍编码效率  
✅ **完成信创集成**：在国产操作系统和数据库环境下部署应用，成功调用大模型API处理JSON数据  
✅ **培养创新思维**：具备代码规范意识，能用AI解决复杂工程问题
